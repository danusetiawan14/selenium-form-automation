# Selenium Form Automation

Professional Python automation project using Selenium.

## Overview

This project demonstrates how to automate online forms using Python and Selenium.

Features include:

- Read customer data from Excel
- Automatically fill online forms
- Browser automation
- Screenshot capture
- Logging
- Error handling
- Export execution reports

---

## Technologies

- Python
- Selenium
- OpenPyXL
- Pandas
- ChromeDriver
- Logging
